Developer Portfolio
Showcase your work, skills, and experience. The portfolio that gets you hired.
▲ v0⚡ Bolt📋 876 copies
Create a developer portfolio: **Hero Section:** - Your name as headline - Role/title (e.g., "Full-Stack Developer") - One-line bio - CTA: "View My Work" + "Contact Me" - Subtle animated background **About Section:** - 2-3 paragraphs about you - Profile photo - Tech stack badges **Projects Grid (6 projects):** - Project thumbnail - Title and one-line description - Tech tags - Links: Live Demo / GitHub **Experience Timeline:** - Company name, role, dates - Bullet points of achievements **Skills Section:** - Categorized: Frontend / Backend / Tools - Skill badges with proficiency level **Contact:** - Email link - Social links (GitHub, LinkedIn, Twitter) - Contact form Dark theme, clean typography, subtle animations.
Minimalist Blog
Clean, typography-focused blog. Perfect for writers and thinkers.
▲ v0⚡ Bolt📋 654 copies
Build a minimalist blog: **Homepage:** - Blog title/logo - Featured post (large card) - Recent posts list (title + date + excerpt) - Load more / pagination **Post Page:** - Title, date, reading time - Author info - Markdown content - Table of contents (sticky) - Share buttons - Related posts **About Page:** - Author bio - Photo - Social links **Design:** - White/off-white background - Serif for body, sans for headings - Generous whitespace - No distractions Use MDX or markdown files for content
Photo Gallery
Visual portfolio with lightbox, albums, and smooth transitions.
▲ v0📋 432 copies
Create a photo gallery: **Albums:** - Grid of album covers - Album title and photo count **Gallery View:** - Masonry or grid layout - Hover: show title - Click: open lightbox **Lightbox:** - Full-size image - Left/right navigation - Image counter - Close on escape or click outside - Swipe support on mobile **Features:** - Lazy loading - Smooth zoom - Download button - Share button Minimal UI, photos are the focus.
Resume / CV Website
Professional resume site. Download as PDF, print-friendly.
▲ v0⚡ Bolt📋 321 copies
Build a resume website: **Header:** - Name, title, location - Contact info row - Download PDF button **Summary:** - Professional summary (3-4 sentences) **Experience:** - Company, role, dates - Achievements as bullet points **Education:** - Degree, school, dates **Skills:** - Categorized skill lists **Projects:** - Key projects with links **Print Styles:** - Clean print layout - Hide interactive elements - Black text on white Clean, professional, easy to update.
